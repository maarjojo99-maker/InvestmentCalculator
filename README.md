# Investment Calculator

A React application to calculate investment returns over time.

## Features
- Real-time calculation based on user input.
- Displays annual data in a tabular format.
- handles initial investment, annual investment, expected return, and duration.

## Core Concepts
- Handling side effects and state synchronization.
- Data validation and error messaging.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
